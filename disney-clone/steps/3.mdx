# Step 3 - Building the channels Section

1. Create a new `section` tag
1. Change the container display to flex
1. Create 5 div elements inside the section for each channel.
1. Create an image tag inside each div, import the respective image for each image tag from `assets/channels` folder
1. Add `border` of #A8A8A8, border-radius, `box-shadow` properties to the `div`
